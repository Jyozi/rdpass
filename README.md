# rdpass
version : 1.1.4

Install
```
git clone https://github.com/Jyozi/rdpass.git
chmod 755 rdpass/rdpass
mv rdpass/rdpass /usr/local/bin/rdpass
```

Usage
```
rdpass [-n Number of password] (option:[-s Accept symbol])
```
